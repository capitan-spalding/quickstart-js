Bienvenido a Quickstart JS

Ejemplo de inicio rápido con Firebase para Web

¿Qué es esto?

Un proyecto de ejemplo para aprender a integrar Firebase en aplicaciones web.

Características principales

  • Autenticación con Firebase
  • Base de datos en tiempo real / Firestore
  • Despliegue fácil con Firebase Hosting o GitHub Pages

Comenzar

  1. Configura tu proyecto Firebase y descarga el archivo de configuración firebaseConfig.
  2. Coloca tu archivo index.html y los archivos necesarios en la carpeta public/ o docs/.
  3. Despliega usando Firebase CLI o GitHub Actions.
